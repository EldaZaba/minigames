# Minihry CZ

Jednoduchý web s minihrami v češtině. Otevřete `index.html` a můžete začít hrát.

## Spuštění
- Otevřete `index.html` v libovolném moderním prohlížeči.

## Seznam her
- Puzzle
- Pexeso (Memory)
- 2048
- Had (Snake)
- Piškvorky (Tic Tac Toe)
- Čtyři v řadě (Connect Four)
- Kámen, nůžky, papír
- Simon říká (Simon Says)
- Šibenice (Hangman)
- Minesweeper
- Flappy Bird
- Breakout
- Whack-a-Mole
- Lights Out
- Tetris

## Motivy
V pravém horním rohu je přepínač motivu (Sunset, Ocean, Forest).
